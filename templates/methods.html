 Part 1 — Training
Architecture

Model design

NORMA is an autoregressive transformer conditioned on longitudinal history, demographics, the target clinical state, and elapsed time to next measurement. It outputs a Gaussian (μ, σ²) for calibrated predictive uncertainty.

Architecture

  • Transformer encoder — {{ model_info.nlayers }} layers, d_model={{ model_info.d_model }}
  • Time2Vec — temporal embedding for irregular intervals
  • State conditioning — Low / Normal / High clinical state token
  • Gaussian NLL head — outputs (μ, σ²) for calibrated uncertainty
  • Test ID embedding — shared across {{ model_info.n_tests }} lab tests
  • Demographic conditioning — age (continuous) and sex (binary)
  • Run ID — {{ model_info.run_id }}

Training setup

  • Training data — EHRSHOT longitudinal EHR sequences
  • External validation — MIMIC-IV (no retraining)
  • Objective — autoregressive next-value prediction
  • Loss — Negative log-likelihood under Gaussian output
  • Normalization — {{ 'per-test z-score' if model_info.normalize else 'raw values, reference-anchored' }}
  • Val R² {{ model_info.val_r2 }}  ·  Val loss {{ model_info.val_loss }}
