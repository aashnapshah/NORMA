Perform a submission readiness check for the manuscript. Target venue: $ARGUMENTS

## Steps

1. Read the manuscript.
2. If a venue is specified, note its requirements (page limits, anonymity, checklist, artifact expectations). For current rules, check official sources rather than relying on memory.
3. Run through the full readiness checklist below.

## Scientific Quality
- [ ] Main claims are explicit and scoped.
- [ ] Every headline claim maps to a figure, table, theorem, or appendix anchor.
- [ ] Strongest relevant baselines are covered.
- [ ] Uncertainty or robustness evidence is included for empirical claims.
- [ ] Limitations are specific and honest.

## Writing Quality
- [ ] Abstract states problem, method, evidence, and takeaway.
- [ ] Introduction states gap and contributions clearly.
- [ ] Related work differentiates against closest prior work.
- [ ] Results section interprets numbers instead of only listing them.
- [ ] Conclusion does not overclaim beyond the evidence.

## Reproducibility and Compliance
- [ ] Data availability statement is present if needed.
- [ ] Code or artifact availability statement is present if needed.
- [ ] Supplement or appendix is navigable.
- [ ] Venue checklist items are complete.
- [ ] Page-limit and anonymity constraints were checked against official sources.

## Final Pass
- [ ] Reviewer-style questions were answered in the paper.
- [ ] Figures and tables have self-contained captions.
- [ ] References include the closest and most recent comparison points.
- [ ] The paper can be summarized accurately in one paragraph.
